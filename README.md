# 📊 YouTube Recommendation Analysis Dashboard

## 📌 Project Overview

This project analyzes YouTube recommendation data to understand user viewing behavior, engagement patterns, and recommendation performance. The analysis focuses on how users interact with recommended videos by examining click-through rates, watch percentage, completion rates, subscription conversions, device usage, content categories, and viewing time.

The project demonstrates the complete data analytics workflow—from data cleaning and transformation to interactive dashboard creation and storytelling using Power BI.

---

## 🎯 Project Objectives

* Analyze user engagement with YouTube recommendations.
* Measure Click-Through Rate (CTR), Watch Percentage, Completion Rate, and Subscription Rate.
* Compare viewer behavior across content categories.
* Analyze engagement across different devices.
* Identify the best time of day for user engagement.
* Study the relationship between video duration and watch behavior.
* Present business insights through an interactive Power BI dashboard.

## 📊 Key Insights

### Recommendation Performance

* Around **30%** of recommendations result in a click.
* Users watch approximately **75%** of videos on average.
* Nearly **50%** of videos are watched until completion.
* Around **5%** of viewers subscribe after watching.

### Content Categories

* Engagement remains consistent across all categories including:

  * Comedy
  * Music
  * News
  * Sports
  * Gaming
  * Lifestyle
  * Education
  * Tech

### Device Analysis

* Mobile, Desktop, Tablet, and TV users show very similar viewing behavior.
* No significant performance differences were observed across devices.

### Time of Day

* Evening and Night sessions generate slightly better engagement than Morning and Afternoon.

### Video Duration

* Longer videos generate more total watch time.
* Completion rates decrease as video duration increases.
